# Cross-Cultural Early Detection of Suicidal Ideation in Adolescents: An Ensemble Machine Learning Approach Using GSHS Data

This repository contains the machine learning code and associated files for the paper "Cross-Cultural Early Detection of Suicidal Ideation in Adolescents: An Ensemble Machine Learning Approach Using GSHS Data".

## Repository Structure

- `ml_codes/`: Contains the machine learning scripts for data preprocessing, model definitions, and training loops.
- `marged_dataset_model_train/`: Contains datasets and models.
- `updates-analysis/`: Analysis for the paper updates.
- `requirements.txt`: Python package dependencies.
- `camera_ready_updates.tex`: The camera-ready version LaTeX files.
